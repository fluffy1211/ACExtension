Troubleshooting

How to check console logs:

  1. Right-click anywhere on the page where you want to use the extension
  2. Select "Inspect" or "Inspect Element"
  3. Go to the "Console" tab
  4. Look for messages that start with "Animal Crossing Typing"

Extension-specific console:

  1. Open a new browser tab
  2. Type edge://extensions (Edge) or chrome://extensions (Chrome)
  3. Find this extension and click "Details"
  4. Scroll down and find "Inspect views: background page"
  5. Click on that to see the background script console
